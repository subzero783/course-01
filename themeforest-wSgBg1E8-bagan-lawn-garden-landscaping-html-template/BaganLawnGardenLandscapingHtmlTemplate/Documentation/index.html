HTML Structure - top

Gardenic HTML Template is built with the Bootstrap 4 framework - that means you have access to all the great features, css & js goodies has to offer. Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases.

To learn more about Bootstrap's grid system, please refer to Bootstrap 4 Grid

CSS Structure - top

I'm using one main CSS file (assets/css/style.css) and sass file available on (assets/sass/style.scss), in this template which you are free to modify depending on the customization you require. There are also other additional style sheets that I do not recommend changing. Those style sheets reside in the 'css' folder and 'sass' file on 'sass' folder.

  • https://fonts.google.com/ - Web font Cabin from Google Fonts
  • bootstrap.min.css - This is a main stylesheet that required for basic template styling like form, input, div, list element and etc.
  • font-awesome.min.css - Web font containing all the icons from the Bootstrap framework and more.
  • style.css - The main stylesheet you are encouraged to modify.

Color Styles (Variations) - top

This theme comes with Unlimited color variations. To change the color style from the theme edit the following lines:

					
						...

						// primary  color list
							$primary-color: #008435; // aqua
							$primary-dark: #004b1e; // aqua
						// secondary color list
							$yallow-greeen: #489f10;
							$yallow-greeen-dark: #3a7d0e;
						// tilte color
							$title-color: #282828;

						// section bg color 
							$section-bg: #f8fafe;
							$section-bg-v2: #f4f4f4;

Just change the color code name for example form $color-primary: #008435; to #004b1e. Available assets/sass/style.scss directory
